Patch? What?

Spotify broke AppleScript support on OS X when they released Spotify version 1.0.1. Crippling apps like Denied, Reign, BarRemote, Spotify Mini Player, Simplify and many more.

How does this work?

This patch fixes the path to the '.sdef' file that OS X needs to enable AppleScript. Check out the source code for more details.

Can I do this manually?

Erm, yea sure.

  1. Right click Spotify in your dock.


  2. Right click Spotify.app and select 'Show Package Contents'.


  3. Go into 'Contents' and open 'Info.plist' in your favorite editor.


  4. Find the key 'OSAScriptingDefinition' and change the value to 'applescript/Spotify.sdef'.

  5. Quit Spotify and start it again.

  6. Repeat for every Spotify update (or just use the patcher).
